find_molecule {biocad_registry} R Documentation

Find or Register a Target Metabolite in BioCAD Registry

Description

This function searches for a metabolite in the BioCAD registry using cross-referencing IDs. If not found, it attempts to register a new metabolite entry with provided metadata.

Usage

find_molecule(biocad.registry, meta, xref.id);

Arguments

biocad.registry

A BioCAD registry connection object representing the database interface.

meta

A list containing metabolite metadata. Expected elements: \itemize{ \item \code{name}: (string) Common name of the metabolite \item \code{formula}: (string) Chemical formula \item \code{description}: (string) Optional descriptive text }

xref.id

(string) Cross-referencing identifier used for database lookup

Details

Authors

xieguigang <xie.guigang@gcmodeller.org>

Value

Returns a metabolite object if found/registered successfully. Returns \code{NULL} if: \itemize{ \item Metabolite not found and failed to register \item Invalid input parameters \item Database errors occur }

Examples

#\dontrun{

result <- find_molecule(
biocad_registry = my_registry,
meta = list(
name = "ATP",
formula = "C10H16N5O13P3",
description = "Adenosine triphosphate"
),
xref_id = "CHEBI:15422"
)

#}
