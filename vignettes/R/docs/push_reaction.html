push_reaction {biocad_registry} R Documentation

Push a single reaction model into registry database

Description

Usage

push_reaction(biocad.registry, reaction);

Arguments

reaction

the external reaction data object

Details

the reaction data model should be a data list in format: r list( entry = "unique_id", definition = "name", comment = "description_string", enzyme = ["ec_number"], db_xrefs = [list(name = "dbname", text = "xref_id")], compounds = [list( side = "string", compound = list(entry,name,formula, factor)) ] ) `

Authors

xieguigang <xie.guigang@gcmodeller.org>

Value

This function has no value returns.
