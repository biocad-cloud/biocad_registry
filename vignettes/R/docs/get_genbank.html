get_genbank {biocad_registry} R Documentation

Download GenBank Assembly File from NCBI FTP Server

Description

This helper function constructs the NCBI FTP path for a given assembly ID, retrieves the corresponding GenBank genomic file (*.gbff.gz), and downloads it to a local directory.

Usage

get_genbank(asm.id,
    repo.dir = './');

Arguments

asm.id

Character. NCBI GenBank assembly accession ID in the format "prefix_Integers" (e.g., "GCF_123456789"). The prefix typically indicates the assembly type (e.g., GCF/GCA), followed by a 9+ digit identifier.

repo.dir

Character. Local directory path to save the downloaded file. Defaults to the current working directory ("./").

Details

The function: \enumerate{ \item Parses the assembly ID into prefix and numeric components \item Constructs the NCBI FTP directory path using the ID structure \item Retrieves the compressed GenBank file (*_genomic.gbff.gz) \item Downloads the file to the specified local directory }

Authors

xieguigang <xie.guigang@gcmodeller.org>

Value

Invisibly returns the full path to the downloaded file. Primarily called for its side effect of downloading the GenBank file to the specified directory.

Examples

#\dontrun{

# Download assembly GCF_123456789 to the current directory
get_genbank("GCF_123456789")
# Save to a custom directory
get_genbank("GCA_987654321", repo_dir = "path/to/genbank_files")

#}
