check_metabolite_synonym {biocad_registry} R Documentation

Internal metabolite synonym checker Helper function that searches for metabolites by name and synonyms when direct xref matching fails. Uses exact mass and MD5 hashing of synonyms for efficient lookup.

Description

Usage

check_metabolite_synonym(biocad.registry, compound);

Arguments

biocad.registry

A database connection object for BioCAD registry

compound

Compound list (see \code{check_metabolite} for structure)

Details

Operates in two phases: \enumerate{ \item Direct name match with mass tolerance \item MD5 hash-based synonym search if name match fails }

Authors

xieguigang <xie.guigang@gcmodeller.org>

Value

An integer database ID if match found by name/synonym, otherwise \code{NULL}.
